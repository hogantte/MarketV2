
# 🛒 MarketV2

## 📌 Proje Durumu
Bu proje henüz tamamlanmamıştır ve geliştirme aşamasındadır.

## 📖 Proje Hakkında
MarketV2, kullanıcıların kendi hesaplarını oluşturup ürün ekleyebildiği, satabildiği ve diğer kullanıcıların ürünlerini inceleyebildiği bir web uygulamasıdır.

## 🧰 Kullanılan Teknolojiler
- HTML
- CSS
- PHP
- JavaScript (eklenmesi planlanıyor)

## 🚀 Aktif Özellikler
- 👤 Kullanıcı kayıt (Register) ve giriş (Login)
- 📦 Ürün ekleme
- 🗑️ Ürün silme
- 📋 Kullanıcıların kendi ürünlerini görüntüleyebilmesi

## 🔮 Planlanan Özellikler
- ✏️ Ürün düzenleme
- 📊 Ürüne stok ekleme
- 🏠 Ürünlerin ana sayfada herkese gösterilmesi
- 🛍️ Ürün satın alma sistemi

## ⚙️ Kurulum
1. XAMPP veya benzeri bir localhost sunucusu kur
2. Projeyi `htdocs` klasörüne at
3. Tarayıcıdan şu adresi aç: http://localhost/marketV2/
